Novus

Make Greate Games in the Browser!

The Novus game engine is an event driven modular game framework for the web. Its power lies in fast iteration using customizable, pluggable components.

Download ZIP File View on GitHub

Getting Started

To get started download the package either through npm, downloading the zip file, or cloning the github repository. From here you can use the command line tool in the bin/ directory to create your project using 

$ novus init <project_name>

This will setup a project structure containing the following folders: 

config/
entities/
models/
renderers/
game.coffee

You can then run $ novus server to start up the server and navigate to http://localhost:8000 to see the project running.

Hello World

The code sets up a basic configuration file for you under the config directory. You can look at this to see what it does on startup and change what engines you want to use.
